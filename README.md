# linkedin_clone

LinkedIn Clone made with flutter.

## ðŸ’» Installation

- You should have updated `Flutter 3.10` now.

Open Terminal and open the location where you want to clone this project and run the following commands.

```
git clone https://github.com/imobasshir/linkedin_clone.git
cd linkedin_clone
flutter create .
flutter pub get
flutter run
```
